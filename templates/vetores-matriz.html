Vetores e matrizes

Vetores (listas) e matrizes (listas de listas) são usados para armazenar coleções de dados. Em Python, os vetores são representados por listas e as matrizes por listas de listas.

Vetores:

Vetores são estruturas de dados que permitem armazenar uma coleção de elementos, como números, strings ou objetos. Eles são úteis para organizar e manipular dados de forma eficiente. Em Python, os vetores são representados por listas, que são coleções ordenadas e mutáveis. Você pode criar um vetor usando colchetes [] e separar os elementos por vírgulas.

Exemplo de vetor:

numeros = [1, 2, 3] Vetor dos números inteiros de 1 a 3.

Neste exemplo, o vetor numeros contém três elementos: 1, 2 e 3. Você pode acessar os elementos do vetor usando índices, começando do zero. Por exemplo, numeros[0] retorna o primeiro elemento (1), numeros[1] retorna o segundo elemento (2) e assim por diante. Veja abaixo a sintaxe de acesso aos elementos do vetor:

print(numeros[0]) # Imprime 1
print(numeros[1]) # Imprime 2
print(numeros[2]) # Imprime 3

Os vetores são úteis para armazenar sequências de dados, como listas de números, nomes ou qualquer outro tipo de informação que você queira agrupar. Você pode adicionar, remover ou modificar elementos do vetor usando métodos como append(), remove() e pop().

numeros.append(4) # Adiciona o número 4 ao vetor
numeros.remove(2) # Remove o número 2 do vetor
numeros.pop() # Remove e retorna o último elemento (3)

Além disso, você pode iterar sobre os elementos do vetor usando estruturas de repetição, como for e while, para realizar operações em cada elemento.

for numero in numeros:
print(numero) # Imprime cada número do vetor

Matrizes:

Matrizes são estruturas de dados que permitem armazenar coleções de elementos em duas dimensões, como tabelas ou grades. Elas são úteis para representar dados organizados em linhas e colunas. Em Python, as matrizes são representadas por listas de listas, onde cada lista interna representa uma linha da matriz. Você pode criar uma matriz usando colchetes aninhados [[]] e separar os elementos por vírgulas.

Exemplo de matriz:

matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]] # Matriz 3x3 com números de 1 a 9.

As matrizes são úteis para representar tabelas, gráficos ou qualquer estrutura bidimensional. Você pode acessar os elementos de uma matriz usando dois índices: o primeiro para a linha e o segundo para a coluna. Por exemplo, matriz[0][1] retorna o elemento da primeira linha e segunda coluna.

print(matriz[0][0]) # Imprime 1 (primeira linha, primeira coluna)
print(matriz[1][2]) # Imprime 6 (segunda linha, terceira coluna)
print(matriz[2][1]) # Imprime 8 (terceira linha, segunda coluna)

As matrizes também podem ser manipuladas usando métodos como append() e remove(), mas lembre-se de que você estará lidando com listas de listas.

matriz[0].append(10) # Adiciona o número 10 à primeira linha da matriz
matriz[1].remove(5) # Remove o número 5 da segunda linha da matriz

Em resumo, vetores e matrizes são estruturas de dados fundamentais em Python que permitem armazenar e manipular coleções de dados de forma eficiente. Eles são amplamente utilizados em programação para resolver uma variedade de problemas, desde simples listas até estruturas mais complexas.
