Estruturas de repetição

Estruturas de repetição permitem executar um bloco de código várias vezes, desde que a sua condição seja verdadeira. Em Python, as principais são for e while.

Estrutura 'for'

for i in range(5):
print(i) # Imprime números de 0 a 4

Neste exemplo, a estrutura de repetição for itera sobre os números de 0 a 4, imprimindo cada número. A função range(5) refere-se ao intervalo de números de 0 a 4. A variável i assume cada valor do intervalo em cada iteração.

A estrutura for é útil quando você sabe quantas vezes deseja repetir um bloco de código, como iterar sobre uma lista ou um intervalo de números.

Estrutura 'while'

contador = 0
while contador < 5:
print(contador)
contador += 1 # Incrementa o contador

Neste exemplo, a estrutura de repetição while continua executando o bloco de código enquanto a condição contador < 5 for verdadeira. A variável contador é incrementada em 1 a cada iteração, garantindo que eventualmente a condição se torne falsa e o loop seja encerrado.

A estrutura while é útil quando você não sabe quantas vezes precisará repetir um bloco de código, como em loops que dependem de condições dinâmicas.

Diferença entre 'for' e 'while'

A principal diferença entre as estruturas de repetição for e while é que o for é usado quando você sabe quantas vezes deseja repetir um bloco de código, enquanto o while é usado quando a repetição depende de uma condição que pode mudar durante a execução do programa.

Em resumo, use for para iterar sobre sequências ou intervalos conhecidos e while para loops que dependem de condições dinâmicas.
