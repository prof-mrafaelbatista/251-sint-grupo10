Estruturas de seleção

Em Python, as estruturas de seleção permitem que o programa execute diferentes blocos de código com base em condições, representadas por expressões lógicas ou relacionais. As principais estruturas de seleção são: if, elif e else.

Estrutura 'if'

A estrutura if é usada para executar um bloco de código se uma condição for verdadeira. A sintaxe é a seguinte:

idade = 20
if idade >= 18:
print("Você é maior de idade.")

Neste exemplo, o programa verifica se a variável idade é maior ou igual (>=) a 18. Se o resultado da expressão lógica for verdadeiro, ele imprime uma mensagem indicando que a pessoa é maior de idade.

Estrutura 'elif'

A estrutura elif (abreviação de 'else if') é usada para verificar múltiplas condições. Se a condição do if for falsa, o programa verifica a condição do elif. A sintaxe é a seguinte:

idade = 16
if idade >= 18:
print("Você é maior de idade.")
elif idade >= 13:
print("Você é adolescente.")

Neste exemplo, o programa verifica se a variável idade é maior ou igual a 18. Como o resultado é falso (idade = 16), ele pula para o bloco do elif e verifica se a idade é maior ou igual a 13. Se essa condição for verdadeira, ele imprime uma mensagem indicando que a pessoa é adolescente.

Estrutura 'else'

A estrutura else é usada para executar um bloco de código se nenhuma das condições anteriores (if ou elif) for verdadeira. A sintaxe é a seguinte:

idade = 10
if idade >= 18:
print("Você é maior de idade.")
elif idade >= 13:
print("Você é adolescente.")
else:
print("Você é criança.")

Neste exemplo, o programa verifica se a variável idade é maior ou igual a 18. Como o resultado é falso (idade = 10), ele pula para o bloco do elif e verifica se a idade é maior ou igual a 13. Como essa condição também é falsa, ele executa o bloco do else, imprimindo uma mensagem indicando que a pessoa é criança.
