Tratamentos de Exceção

Tratamentos de exceção são usados para lidar com erros que podem ocorrer durante a execução do programa, como divisão por zero, acesso a índices fora do intervalo de uma lista ou erros de tipo. Em Python, você pode usar as palavras-chave try e except para capturar e tratar exceções.

O bloco try contém o código que pode gerar uma exceção, enquanto o bloco except define como lidar com a exceção caso ela ocorra.

Exemplo de tratamento de exceção:

try:
resultado = 10 / 0 # Isso gera uma exceção de divisão por zero
except ZeroDivisionError:
print("Erro: Divisão por zero não é permitida.")

Neste exemplo, o código dentro do bloco try tenta realizar uma divisão por zero, o que gera uma exceção ZeroDivisionError. O bloco except captura essa exceção e imprime uma mensagem de erro personalizada.

Você também pode capturar exceções genéricas usando Exception para lidar com qualquer tipo de erro:

try:
resultado = 10 / 0
except Exception as e:
print("Ocorreu um erro:", e)

Neste exemplo, qualquer exceção que ocorra dentro do bloco try será capturada pelo bloco except, e a variável e conterá a mensagem de erro.

O tratamento de exceções é importante para garantir que o programa continue funcionando mesmo quando ocorrem erros inesperados. Ele permite que você forneça mensagens de erro amigáveis ao usuário e evite que o programa seja encerrado abruptamente.
